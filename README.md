# elysia_bun

To install dependencies:

```bash
bun install
```

To init prisma:

```bash
bun prisma init
```


To generate prisma model :

```bash
bun prisma generate
```

To create migration:

```bash
bun prisma db push
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.1.16. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
